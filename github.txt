git clone https://github.com/*.git
Клонирует репозиторий.

git status
Показывает статус изменений.

git status -s
Кратко показывает статус изменений.

git clean -f
Принудительная очистка (удаление новых файлов, которых нет в репо).

git reset
Удаления всех новых локальных коммитов.

git reset --hard
Откат всех локальных изменений до последнего коммита. 

git commit --amend
Перезаписать последний коммит.

git clone https://github.com/*.git
Клонирование существующего репозитория.

git reset --hard a3775a5485af0af20375cedf46112db5f813322a 
git push --force
Удалить коммиты.

git diff 
Выводит изменения через консоль.

git difftool
Выводит изменения в GUI по одному файлу.

git difftool --dir-diff
Открывает список измененных файлов в GUI.
